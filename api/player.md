# player.getPosArray
Gets the position of the player as a array
```lua
    player.getPosArray() -> [x,y,z]
```
# player.getPlayerName

Returns the name of the player.
```lua
player.getPlayerName()
```

# player.getX
:::warning
Deprecated, use getPosArray instead
:::
Returns player's X-coordinate.
```lua
player.getX()
```
